## Implementation Priority

### High Priority (Easy to Implement)
- **Badge Hunter** - Count unlocked achievements
- **Arena Elite** - Leaderboard position check
- **Iron Enduro** - Check journey distance
- **Performance** - Compare current vs personal best stats

### Medium Priority (Requires New Tracking)
- **Ace of Friends** - Need referral/invite system
- **Drift Detection** - Need accelerometer/gyroscope analytics
- **Night Journeys** - Need time-of-day tracking
- **Vehicle Types** - Need vehicle categorization

### Low Priority (Complex Features)
- **Route Tracking** - Need geofencing/route comparison
- **City Tracking** - Need reverse geocoding
- **Acceleration Tracking** - Need high-frequency GPS/sensor data
- **Race Mode** - Need new journey mode implementation
